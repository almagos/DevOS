# Purpose

Linear is the Source of Truth for planning within DevOS.

Parent Issues and Child Issues are authored in Linear before implementation begins.

---

# Responsibilities

Linear is responsible for:

- Planning
- Parent Issues
- Child Issues
- Prioritization
- Work organization

Linear is not responsible for:

- Source code
- Implementation
- Repository documentation
- Knowledge Synchronization
