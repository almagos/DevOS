# Integrations

## Purpose

Details external system integrations, third-party APIs, and product ecosystem touchpoints.

## Contains

- High-level descriptions of third-party platform integrations.
- Data exchange contracts and external partner capabilities.
- Integration scopes and functional boundaries.

## Does Not Contain

- API secret keys, passwords, or production credentials.
- Low-level SDK implementation code.
- Temporary testing endpoints or mock tokens.
