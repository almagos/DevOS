# Users

## Purpose

Documents user personas, target user groups, user roles, and core user workflows.

## Contains

- Primary and secondary user personas.
- Key user journeys and interaction profiles.
- Role-based permission expectations and user needs.

## Does Not Contain

- Personal Identifiable Information (PII) or user telemetry data.
- Specific auth code or token management implementation.
- Temporary user feedback or raw survey outputs.
