# Domain

## Purpose

Captures core domain models, business terminology, bounded contexts, and domain concepts.

## Contains

- Ubiquitous language dictionary and domain concepts.
- Business entities, value objects, and relationship descriptions.
- Bounded context definitions.

## Does Not Contain

- Database schemas or ORM mapping code.
- UI component state details.
- Infrastructure and platform configurations.
