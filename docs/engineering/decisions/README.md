# Decisions

## Purpose

Houses Architectural Decision Records (ADRs) capturing significant technical and architectural choices.

## Contains

- Formally adopted Architectural Decision Records (ADRs).
- Context, rationale, and consequences of key technical decisions.
- Superseded ADR references and decision history.

## Does Not Contain

- Active technical debate or unapproved proposals.
- Minor daily code review comments.
- General product strategy notes.
