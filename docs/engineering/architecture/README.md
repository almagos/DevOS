# Architecture

## Purpose

Defines technical design, system components, subsystem relationships, and data flow patterns.

## Contains

- High-level architectural diagrams and overview documents.
- Subsystem descriptions and boundary definitions.
- Cross-cutting technical specifications.

## Does Not Contain

- Individual function-level docstrings or code comments.
- Operational incident logs.
- Outdated legacy architecture notes.
