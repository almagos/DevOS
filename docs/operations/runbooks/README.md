# Runbooks

## Purpose

Provides actionable, step-by-step guides for diagnosing, mitigating, and resolving operational incidents.

## Contains

- Incident response protocols and triage procedures.
- Service recovery steps and failover execution instructions.
- Common failure mode resolution runbooks.

## Does Not Contain

- Post-mortem incident analysis archives.
- General software design documentation.
- Plaintext production credentials.

## Related Directories

- `docs/operations/monitoring/`
- `docs/playbooks/`

- [docs/operations/README.md](../README.md)

## Naming Conventions

Use lower-case kebab-case with descriptive names (e.g., `database-failover.md`).
