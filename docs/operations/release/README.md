# Release

## Purpose

Documents release procedures, deployment workflows, rollback policies, and versioning standards.

## Contains

- Release verification checklists.
- Deployment step sequencing and release cadence guidelines.
- Rollback strategies and emergency patch protocols.

## Does Not Contain

- Individual commit release notes (generated by CI/tagging).
- Feature specification documents.
- Live environment secret values.
