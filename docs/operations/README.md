# Operations

## Purpose

Contains operational knowledge regarding deployment releases, system monitoring, and runbooks.

## Contains

- `release/`: Deployment procedures, release checklist, and versioning guides.
- `monitoring/`: Observability strategy, metrics, logging, and alert rules.
- `runbooks/`: Step-by-step incident response and operational troubleshooting guides.

## Does Not Contain

- Source code implementation.
- Product strategy or feature roadmaps.
- Raw log dumps or historical metric exports.

## Related Directories

- `docs/engineering/platform/`
- `docs/playbooks/`
